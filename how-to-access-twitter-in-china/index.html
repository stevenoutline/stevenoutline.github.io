推特是什么，国内如何使用

发表于 2023-08-22 阅读次数：

推特（Twitter）是全球最著名和使用广泛的社交媒体平台之一，旨在帮助人们与世界分享和发现新闻、信息和兴趣。在推特上，用户可以发布 140 个字符的短消息，即”推文”（Tweets），并与其他用户进行互动。

很多人可能会好奇，在国内是否能够使用推特。实际上，由于一些特殊的原因，包括政策和互联网审查，推特在中国大陆被墙，因此无法直接访问和使用。然而，国内的用户仍然可以通过一些方式来绕过这个限制。

首先，你可以使用虚拟私人网络（VPN）来访问推特。VPN 是一种加密的网络连接方式，可以通过连接到位于国外的服务器来绕过网络限制和审查。通过设置 VPN 并连接到境外服务器，你可以访问和使用推特，就像你身处其他国家一样。

如何下载 Twitter App

苹果手机用户可以通过 APP Store 进行下载，只需在 App Store 下搜索 “推特” 就能搜索到, 不过现在推特换图标了, 小蓝鸟换成了 X。

如何注册 Twitter 账号

  1. 打开 twitter 官网 https://twitter.com/
  2. 按照提示填写相关信息,出生日期最好填大于 18 岁，不然限制很多你想看的内容
  3. 电话号码可以是国内的，记得加上 +86
  4. 如果手机收不到验证码，可以使用邮箱注册。

Twitter 使用教程

设置个人资料

  • 点击个人头像或者左侧 “profile”
  • 进去之后，点击 “Set up profile”
  • 开始设置你的头像，对自己做一个简单的描述，以及设置位置。

关注用户

你可以通过输入关键字或者名字全称，然后 “follow” 他

发推文

这个也很简单，进入首页你就可以看到一个可以输入文字的边框，在里面输入内容，然后点击 “tweet” 即可。

转发推文

对于别人的推文，你可以转发，在推文的底部有一些选项：喜欢，回复，转发等

发送私信

点击个人主页顶端的私信标签。屏幕上会弹出 ”私信” 窗口。进入输入内容，发送即可。
