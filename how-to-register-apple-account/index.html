如何注册美国区的苹果账号

发表于 2023-06-06 阅读次数：

大多数时候，国区的 Apple ID 基本上可以满足大多数国内用户的所有需求。不过如果你想要下载一些外区 Apple Store 才上架的 App (比如. Shadowrocket, PUBG Mobile, TikTok, LINE 等)，那就需要使用外区的 Apple ID 登录外区的 App Store。本教程将以注册美区 Apple ID 为例，详细介绍如何免费注册一个外区 Apple ID。

创建 Apple ID

点击 Apple ID 创建链接 https://appleid.apple.com/account（注意不要使用 vpn 等代理，否则无法打开），然后点击 “创建您的 Apple ID”。（页面语言是中文英文都可以）

输入姓名、生日、电子邮件地址、密码、国家选择美国

选择安全提示问题，点击“继续”。

网页版注册完成。

手机登陆新创建的 Apple ID

  1. 打开 Apple 设备，进入 App Store -> 点击右上角你的 Apple ID 头像退出原来的账户。
  2. 填写新创建等 Apple ID -> 点击 “Sign in” 登录。
  3. 此时会跳出验证信息点击 “Review”。
  4. 确认国家是 “United States” 美国 -> 打开 “Agree to Terms and Conditions” 同意条款和条件 -> 点击 “Next” 继续。
  5. 付款方式选择 “None”，点击 “Next” 继续。
  6. 填写姓名，美国地址，如果您没有地址建议网上搜索美国个人信息生成，填写完点击 “Next” 继续。
  7. 点击 “Continue” 继续，完成最终注册。
