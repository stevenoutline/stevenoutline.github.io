2023 如何在国内使用 Facebook (脸书)

发表于 2023-09-16

Facebook(脸书)作为全球第一大社交平台，拥有几十亿全球用户，用户黏性非常高，而且用户质量也非常好，很多国内的朋友也都希望尝试 Facebook。可是遇到第一个问题，那就是：如何注册并拥有一个 Facebook 账户？

首先解决网络问题

Facebook 的主页网址是 http://facebook.com，可是通常情况下我们是无法打开这个网址的, 会提示无法访问此网站.

造成这样的状况并不是电脑问题，而是 Facebook 本身就无法在中国大陆地区访问，想解决这个问题，就必须用到科学上网，至于如何科学上网，可以看我的另外一篇文章 如何使用 Outline VPN。

当我们科学上网之后，就可以打开 Facebook 的主页了，如图显示：

注册前准备

a.注册时候可以用手机号注册，也可以用邮箱注册，大陆手机号是可以注册 Facebook 的，邮箱的话尽量使用国际邮箱，比如 谷歌邮箱 Gmail，邮箱尽量自己注册，因为这个邮箱很重要，后期如果要接码，邮箱死掉了，Facebook 也就无法登录了。如何注册 Gmail 邮箱, 可以看这里 2023 最新注册 Gmail 邮箱详细指南；

b.注册信息，最好是个人的信息，但只需要姓名，性别即可，其他的都不重要，Facebook 并不像国内社交平台，并不需要实名认证，假如你有多个手机号或者多个邮箱，是可以注册很多个的。

c.一个稳定干净的 VPN 是注册成功的关键因素, 这也是我推荐 Outline VPN 的原因.注册全过程都需要开启 Outline VPN 并保持连接状态.

d.推荐使用 Chrome 谷歌浏览器,或者是 Microsoft Edge 微软浏览器.

注册 Facebook 账号

接下来，我们来进入实操环节，这里我用邮箱注册，手机注册同理：

第 1 步：打开 Facebook 官网 点击 新建账户

第 2 步：分别输入 姓、名、邮箱、密码、生日、性别

第 3 步：点击注册，此时跳转下一个页面，会让你输入邮箱验证码

第 4 步：我们去邮箱里找到验证码

第 5 步：然后复制验证码到 facebook，点击继续

第 6 步：点击继续以后，会提示注册成功,点击确定，即可 进入 facebook 主页，注册成功；

Facebook 使用

可以直接通过浏览器的方式使用 Facebook 也可以下载 Facebook App. 苹果手机可以在 App Store 直接搜索 Facebook, 并下载.

前往你的 Facebook 个人主页

桌面版和移动版的 Facebook 主页有些许不同：

桌面版：点击窗口右上方带有你名字的选项卡。

移动版：轻触屏幕右上角或右下角的 ☰，然后在出现的菜单顶端轻触你的名字。

添加个人头像

在个人主页添加一张自己或其它东西的照片，好让其他用户认得你：

桌面版：点击 Facebook 个人主页左上方的添加照片，点击上传照片，从电脑选择一张照片，然后点击打开。

移动版：轻触页面顶端的方形头像图标，轻触选择头像，轻触你要使用的照片，然后轻触使用。

你也要可以在个人主页顶端添加封面照片。只需要点击或轻触添加封面照片，然后点击上传照片(桌面版)，或轻触更改封面照片(移动版)，从电脑或移动设备选择一张照片就可以了。

编辑帐户信息

如果你在创建 Facebook 帐户的时候没有添加到特定信息，或者想要删除之前添加的信息，可以在个人主页这么做：

桌面版：点击封面照片底下的简介，在页面左边的“简介”标题底下选择一个主题，比如你生活过的地方，将鼠标悬停在上面，在编辑选项出现时点击一下，然后进行编辑。

移动版：往下滚动，轻触“分享新鲜事”文本框上方的简介，轻触某个主题右边的铅笔图标，轻触编辑选项，然后进行编辑。

定位好友

我们先加一些圈内的好友，而且最好在圈内有一定人脉。然后查看他们的好友，进而一步一步扩大你的好友圈。

如果你的目标人群不是很集中，不适合第一种方式的话，那就告诉你另一个方法：比如我们定位于音乐发烧友。第一步，先确定国际上最有名的音乐院校有那些，那么在 Facebook 的好友检索中，按照这些条件搜索出来的几乎都是音乐爱好者了。

吸引别人主动加你,比如说,如果我们的目标定位是明星粉丝，你可以加一些明星的粉丝社区，用明星的名字注册 ID，并且在社区里保持一定的活跃性，自然就有那个明星的粉丝加你了。

个人资料完善后，我们就可以在首页中发布动态， 文本、图片、视频、链接都可分享, 这样会吸引和你有共同爱好的朋友。
